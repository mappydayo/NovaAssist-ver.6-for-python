# NovaAssist ver.6 for python

これは python-dotenv と discord.py を使用した Discord Bot です。

## 前提条件

このプロジェクトを実行するには以下が必要です。

* Python 3.12 以上
* uv（Pythonパッケージマネージャー）

## セットアップ手順

1. 依存関係のインストール

プロジェクトのルートディレクトリで以下のコマンドを実行し、仮想環境を作成して依存関係をインストールします。

uv sync

2. 環境変数の設定

.env ファイルを開き、必要な環境変数を設定します。

* DISCORD_BOT_TOKEN: Discord Developer Portalから取得したBotのトークンを設定します。
* GUILD_ID: スラッシュコマンドを即時反映させたい特定のDiscordサーバーのIDを入力します（任意）。

## 起動方法

以下のコマンドを実行してBotを起動します。

uv run main.py

## コマンド一覧

このBotは以下のスラッシュコマンドを提供します。

### /ping

Botの動作確認用コマンドです。実行すると「pong」と応答します。

### /timer

指定した時間経過後にメンション付きで通知するタイマーコマンドです。

* minutes: タイマーをセットする時間（分単位。0.5のように小数で秒単位の指定も可能）
* title: タイマーのタイトル

### /timer_status

実行中のタイマーの残り時間を確認します。現在アクティブなタイマーの一覧とそれぞれの残り時間が分単位で表示されます。
